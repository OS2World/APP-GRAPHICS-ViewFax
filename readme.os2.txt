viewfax 2.5
-----------

A simple fax viewer (G3, G4) for X11.
Further information and manual is located in subdirectory SRC.

Minimum Requirements:
 * OS/2 Warp 3
 * EMX Runtime 0.9d
 * XFree86/2 3.3.6

Installation:
 * copy XFREE86 subdirectory to your X11ROOT drive

Usage:
 > viewfax				 // prints parameters
 > viewfax fax20051201.tif	   	 // open a received fax 
 > viewfax \xfree86\lib\X11\viewfax.tif  // view online help
